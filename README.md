# CrazyKarts: Rocket Rampage 🚀

A 3D zombie survival driving game with rocket combat, developed using Three.js and TypeScript. Drive through a procedurally generated world, mow down zombies, and blast them with rockets.

## 🎮 Game Features

- **Procedurally Generated World**
  - Infinite terrain with dynamic chunk loading
  - Forests, trees, and environmental features
  - Terrain-following vehicle physics

- **Zombie Survival**
  - Zombies chase and swarm the player
  - Run them over at high speed or blast them with rockets
  - Satisfying slime effects when zombies are destroyed
  - Automatic respawning keeps the action going

- **Rocket Combat**
  - Fire rockets from your vehicle
  - Realistic projectile physics with velocity inheritance
  - Explosive impacts and particle effects
  - Ignite trees with rocket hits

- **Vehicle Physics**
  - Realistic acceleration and deceleration
  - Dynamic steering with wheel rotation
  - Responsive camera that follows the action

## 🛠️ Technical Stack

- Three.js for 3D graphics
- TypeScript for type-safe code
- Vite for fast development
- Tweakpane for runtime debugging
- Procedural world generation with chunk-based terrain
- Physics-based movement and collision detection

## 🚀 Getting Started

```bash
pnpm install
pnpm dev
```

## 🎮 Controls

- **W/↑** - Accelerate
- **S/↓** - Brake/Reverse
- **A/←** - Turn Left
- **D/→** - Turn Right
- **Space** - Fire Rocket
